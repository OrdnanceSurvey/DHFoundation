DHNotificationStore Class Reference

Inherits from NSObject
Declared in DHNotificationStore.h

Overview

An object for making NSNotificationCenter’s block APIs a little easier to deal with by automatically handling the observer objects returned.

NOTE: There is no need to store and remove the notification objects externally. this is handled for you automatically in this class. Return values should be used for removeObserver: purposes only when you need to remove the observer before the owner of this object gets deallocated.
