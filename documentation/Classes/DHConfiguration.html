Inherits from NSObject

Overview

Base class for retrieving properties from a config file. Supports properties of types supported by plist (NSNumber, NSString, NSDictionary, NSArray), as well as conversions to BOOL, NSInteger and float. For other conversions, use the extension functionality documented below.

propertyPrefix

Optional prefix for properties in the config file. This class will check for the property name first, and if not found will capitalise the first letter of the property and add this prefix and try again.
